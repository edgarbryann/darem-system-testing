# darem-system-testing
Case study system for Software Testing and Quality Assurance

## Purpose
This system is provided for academic use in Software Testing and Quality Assurance.

Students are required to:
- Perform manual testing
- Create test plans
- Write test cases
- Execute test cases
- Report defects

## Scope
- No code modification required
- Focus on black-box and exploratory testing

## Deliverables
- Manual Test Plan
- Test Case Documentation
- Defect / Bug Reports

## Tools Required
Students must install the following before starting:
- Visual Studio Code
- Node.js (LTS version)
- Git
- MySQL or MariaDB
- HeidiSQL (recommended DB client)

No additional VS Code extensions are required.
